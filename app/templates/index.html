Projeto de Cadastro de Usuários

Este projeto foi desenvolvido com o objetivo de demonstrar habilidades práticas em desenvolvimento web utilizando as seguintes tecnologias:

  • Linguagem: Python
  • Framework: Flask
  • Banco de Dados: SQLite
  • Template Engine: Jinja2
  • Estilização: Bootstrap 5

Para acessar a lista de usuários cadastrados e utilizar as funções de editar e excluir usuários, é necessário primeiro realizar um cadastro e depois efetuar o login.

Importante: Os dados utilizados no cadastro devem ser fictícios, pois este sistema é apenas para fins de demonstração e aprendizado.
